## Grameta Precision Craft Website

Marketing website for UAB Grameta. Built with Vite, TypeScript, React, shadcn-ui, and Tailwind CSS.

### Requirements
- Node.js 18+
- npm 9+

### Install
```sh
npm install
```

### Develop
```sh
npm run dev
```

### Build
```sh
npm run build
```

### Preview build
```sh
npm run preview
```
